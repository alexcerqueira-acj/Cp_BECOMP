# 🎮 Recanto Gamer
Este projeto foi desenvolvido como um trabalho prático para aplicar e consolidar os conhecimentos básicos de Desenvolvimento Web, com foco em estruturação semântica e estilização.

## 🎯 Objetivo do Projeto
O objetivo principal deste site é criar um espaço dedicado aos amantes de jogos, apresentando notícias, informações sobre os principais consoles e uma galeria de imagens responsiva. O foco técnico do projeto foi o domínio das tags HTML5 e a organização do layout utilizando CSS Flexbox.

## ✨ Funcionalidades
O site é composto por três páginas interligadas:
* **Página Inicial (`index.html`):** Apresentação do projeto, missão do site e listas (ordenadas e não ordenadas) com os principais consoles do mercado.
* **Galeria de Jogos (`galeria.html`):** Uma galeria de imagens de jogos organizada e responsiva utilizando o poder do CSS Flexbox.
* **Cadastro (`cadastro.html`):** Um formulário completo para captação de dados de usuários, utilizando diferentes tipos de inputs (texto, email, password, radio e checkbox).

## 🛠️ Tecnologias Utilizadas
* **HTML:** Estruturação semântica de todo o conteúdo (`<header>`, `<main>`, `<section>`, `<nav>`, etc).
* **CSS:** Estilização visual, cores, fontes e organização de layout.
* **Flexbox:** Utilizado especificamente para a construção da galeria de imagens de forma responsiva e alinhada.
* **CSS3:** Estilização visual, cores, fontes e organização de layout.
* **Flexbox:** Utilizado especificamente para a construção da galeria de imagens de forma responsiva e alinhada.
