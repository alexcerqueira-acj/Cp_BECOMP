# 🎮 Recanto Gamer
Este projeto foi desenvolvido como um trabalho prático para aplicar e consolidar os conhecimentos básicos de Desenvolvimento Web, com foco em estruturação semântica e estilização.

## 🎯 Objetivo do Projeto
O objetivo principal deste site é criar um espaço dedicado aos amantes de jogos, apresentando notícias, informações sobre os principais consoles e uma galeria de imagens responsiva. O foco técnico do projeto foi o domínio das tags HTML5 e a organização do layout utilizando CSS Flexbox.

## ✨ Funcionalidades
O site é composto por três páginas interligadas:
* **Página Inicial (`index.html`):** Apresentação do projeto, missão do site e listas (ordenadas e não ordenadas) com os principais consoles do mercado.
* **Galeria de Jogos (`galeria.html`):** Uma galeria de imagens de jogos organizada e responsiva utilizando o poder do CSS Flexbox.
* **Cadastro (`cadastro.html`):** Um formulário completo para captação de dados de usuários, utilizando diferentes tipos de inputs (texto, email, password, radio e checkbox).

## 🛠️ Tecnologias Utilizadas
* **HTML:** Estruturação semântica de todo o conteúdo (`<header>`, `<main>`, `<section>`, `<nav>`, etc).
* **CSS:** Estilização visual, cores, fontes e organização de layout.
* **Flexbox:** Utilizado especificamente para a construção da galeria de imagens de forma responsiva e alinhada.

#ENTREGÁVEL 2

💻 MS Tech Store

Este projeto foi desenvolvido como um trabalho prático para aplicar e aprofundar os conhecimentos em Desenvolvimento Web, com foco em criação de layouts avançados e design de interface de alto padrão.

🎯 Objetivo do Projeto

O objetivo principal deste site é reproduzir o design elegante e minimalista de uma loja de tecnologia premium (com estética inspirada na Apple). O foco técnico absoluto do projeto foi a arquitetura de layout, utilizando a abordagem "Macro para Micro", combinando o rigor estrutural do CSS Grid com a maleabilidade do CSS Flexbox, além da aplicação de efeitos visuais e animações profissionais.

✨ Funcionalidades

O site é composto por cinco páginas totalmente interligadas:

Página Inicial (index.html): Uma vitrine cinematográfica apresentando as grandes áreas da loja, utilizando imagens com efeitos avançados de transição (mudança suave de preto e branco para colorido com zoom sutil via hover).

Páginas de Produtos (smartphones.html, computadores.html, fones.html, acessorios.html): Cada categoria possui sua própria página, onde os itens são exibidos em uma matriz perfeita e centralizada (3 linhas por 2 colunas). Os cards de produtos contêm imagens não-distorcidas, descrições e botões de compra animados.

Navegação Premium: Um cabeçalho com texto em gradiente bordado e uma barra de navegação alinhada horizontalmente que destaca a página ativa de forma dinâmica.

🛠️ Tecnologias Utilizadas

HTML: Estruturação semântica limpa da loja virtual (uso correto de <header>, <nav>, <main> e containers).

CSS: Estilização visual robusta com foco em Dark Mode (fundo escuro e tipografia clara), manipulação de proporção de imagens (aspect-ratio), propriedades de filtro (grayscale) e transições físicas suaves (transform, cubic-bezier).

CSS Grid: Protagonista do projeto. Utilizado obrigatoriamente para construir o esqueleto macro da página e para garantir a distribuição perfeitamente simétrica da grade principal de produtos (Matriz 3x2).

Flexbox: Utilizado para o micro-layout, sendo responsável por organizar a barra de navegação perfeitamente espaçada e empilhar internamente os textos e botões dentro dos cards de cada produto.
