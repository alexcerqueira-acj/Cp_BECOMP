# 🎮 Recanto Gamer
Este projeto foi desenvolvido como um trabalho prático para aplicar e consolidar os conhecimentos básicos de Desenvolvimento Web, com foco em estruturação semântica e estilização.

## 🎯 Objetivo do Projeto
O objetivo principal deste site é criar um espaço dedicado aos amantes de jogos, apresentando notícias, informações sobre os principais consoles e uma galeria de imagens responsiva. O foco técnico do projeto foi o domínio das tags HTML5 e a organização do layout utilizando CSS Flexbox.

## ✨ Funcionalidades
O site é composto por três páginas interligadas:
* **Página Inicial (`index.html`):** Apresentação do projeto, missão do site e listas (ordenadas e não ordenadas) com os principais consoles do mercado.
* **Galeria de Jogos (`galeria.html`):** Uma galeria de imagens de jogos organizada e responsiva utilizando o poder do CSS Flexbox.
* **Cadastro (`cadastro.html`):** Um formulário completo para captação de dados de usuários, utilizando diferentes tipos de inputs (texto, email, password, radio e checkbox).

## 🛠️ Tecnologias Utilizadas
* **HTML:** Estruturação semântica de todo o conteúdo (`<header>`, `<main>`, `<section>`, `<nav>`, etc).
* **CSS:** Estilização visual, cores, fontes e organização de layout.
* **Flexbox:** Utilizado especificamente para a construção da galeria de imagens de forma responsiva e alinhada.

💻 MS Tech Store

Este projeto foi desenvolvido como um trabalho prático para aplicar e aprofundar os conhecimentos em Desenvolvimento Web, com foco em criação de layouts avançados e design de interface de alto padrão.

🎯 Objetivo do Projeto

O objetivo principal deste site é reproduzir o design elegante e minimalista de uma loja de tecnologia premium (com estética inspirada na Apple). O foco técnico absoluto do projeto foi a arquitetura de layout, utilizando a abordagem "Macro para Micro", combinando o rigor estrutural do CSS Grid com a maleabilidade do CSS Flexbox, além da aplicação de efeitos visuais e animações profissionais.

✨ Funcionalidades

O site é composto por cinco páginas totalmente interligadas:

Página Inicial (index.html): Uma vitrine cinematográfica apresentando as grandes áreas da loja, utilizando imagens com efeitos avançados de transição (mudança suave de preto e branco para colorido com zoom sutil via hover).

Páginas de Produtos (smartphones.html, computadores.html, fones.html, acessorios.html): Cada categoria possui sua própria página, onde os itens são exibidos em uma matriz perfeita e centralizada (3 linhas por 2 colunas). Os cards de produtos contêm imagens não-distorcidas, descrições e botões de compra animados.

Navegação Premium: Um cabeçalho com texto em gradiente bordado e uma barra de navegação alinhada horizontalmente que destaca a página ativa de forma dinâmica.

🛠️ Tecnologias Utilizadas

HTML: Estruturação semântica limpa da loja virtual (uso correto de <header>, <nav>, <main> e containers).

CSS: Estilização visual robusta com foco em Dark Mode (fundo escuro e tipografia clara), manipulação de proporção de imagens (aspect-ratio), propriedades de filtro (grayscale) e transições físicas suaves (transform, cubic-bezier).

O objetivo principal deste site é construir uma interface de e-commerce completa, funcional e de alto padrão visual para uma marca de chinelos premium. O foco técnico absoluto do projeto foi a integração entre o design responsivo (utilizando uma paleta sofisticada em Bordô e Bege) e a interatividade lógica com JavaScript, englobando a manipulação de dados em memória, filtros de busca em tempo real e a persistência de estados no navegador.  ✨ FuncionalidadesO site é composto por três páginas totalmente integradas e responsivas:Página Inicial (index.html): Apresenta a identidade da marca e conta com um slider (carrossel) de imagens automatizado e manual através de botões direcionais. O banner utiliza um sistema de renderização inteligente que se adapta sem cortes a diferentes proporções de imagem.  Lojinha (loja.html): Uma vitrine dinâmica onde a grade de produtos é injetada diretamente via JavaScript através de uma base de dados interna. Conta com um sistema de busca em tempo real que filtra os itens instantaneamente por modelo ou por tags associadas, sem necessidade de recarregar a página.  Página de Acesso (login.html): Um sistema de autenticação completo que alterna dinamicamente entre as telas de "Login" e "Registro" na mesma interface. Utiliza regras de validação para impedir e-mails duplicados e gerencia o estado da sessão do usuário.  Efeito Visual de Conversão: Alinhado às técnicas modernas de páginas de vendas, os cards de produtos e os elementos do slider contam com um efeito de transição fluido via hover, que altera a imagem de preto e branco para colorido, acompanhado de um zoom óptico sutil.  🛠️ Tecnologias UtilizadasHTML: Estruturação semântica avançada de todas as interfaces, fazendo uso correto de tags estruturais (<header>, <main>, <section>, <nav>) e de formulários para captura de dados de forma segura.  CSS: Arquitetura visual baseada em variáveis de estilização (:root) para consistência da paleta de cores. Aplicação de filtros físicos (grayscale), manipulação de escala (transform: scale) e responsividade avançada através de Media Queries para garantir adaptabilidade perfeita em dispositivos móveis.  CSS Grid: Utilizado obrigatoriamente para construir a distribuição fluidamente simétrica e automatizada da grade principal de produtos na vitrine, adaptando-se sem quebras à quantidade de itens existentes.  Flexbox: Utilizado para o micro-layout, sendo responsável por organizar a barra de navegação perfeitamente espaçada, alinhar os elementos estruturais do slider e empilhar internamente os textos e botões dentro dos cards de cada produto.  JavaScript (ES6): Responsável por toda a inteligência e dinamismo do projeto, utilizando manipulação do DOM, controle assíncrono de tempo (setInterval para o carrossel), métodos de array (filter e includes para o motor de busca) e persistência de dados localizados através da API localStorage para o fluxo de login.
